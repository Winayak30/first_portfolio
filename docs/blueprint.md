# **App Name**: Cyber Minimalist Portfolio

## Core Features:

- Hero Section: Displays developer's name, bio, and a call-to-action.
- Bento Grid Layout: Organizes content into a visually engaging and scannable grid.
- Glassmorphism Panels: Applies a frosted glass effect to UI containers for a modern look.
- Project Cards: Showcases projects with images/videos, titles, and tech stack tags.
- Skills and Tech Stack: Presents a list of developer's skills and technologies in an organized manner.
- Contact and Socials: Provides contact information and links to social profiles.
- Interactive Orb Cursor: Replaces the standard mouse cursor with a customizable orb that visually responds to on-screen UI elements.

## Style Guidelines:

- Primary background: Deep, near-black space blue (#0A0A0A).
- Glass panel background: Semi-transparent with an alpha value rgba(23, 24, 33, 0.6).
- Glass panel border: Subtle, light border rgba(255, 255, 255, 0.1).
- Vibrant accent: Electric Amethyst (#7F00FF) for interactive elements and highlights.
- Primary text: Soft off-white (#EAEAEF) for body copy and main text.
- Secondary text: Muted grey (#888994) for secondary information and placeholders.
- Headline font: 'Avenir' (sans-serif) for a clean, architectural feel.
- Body font: 'Roboto Mono' (monospace) for a code-focused typographic environment.
- Bento Grid layout for a modern, engaging content organization.
- Custom cursor with orb effect for tactile interactivity.
- Scroll-driven animations to reveal content dynamically.
- 3D Tilt hover effect on project cards for depth and interactivity.
